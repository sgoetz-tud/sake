Detailed Description

This class represents the world being simulated. It is the main class in what concerns the simulation model. It contains all the agents, places and context overlays. This is the class you'll be coming to when writing your World, Agent and Context models.

The methods in this class let you interact with the world, find agents, places, etc...

Author
Miquel Martin
