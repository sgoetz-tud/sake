Detailed Description

An overlay is in essence a matrix which maps a value to each position of the simulated world. Depending on the overlay type, the value can be a boolean value, a discrete value, or others, wrapped in the Publishable format.

An overlay can be created from an image provided in the simulation data, or manually by setting the values of the matrix.
