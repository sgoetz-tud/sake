static String getNextName ()
 

Detailed Description

Utility class to generate unique names. The first 1000 belong to several international lists of hurricane names. After that, the returned names are of the format "Person-#" where # is a number starting at 1.

The names are generated in the same sequence everytime the class is loaded.
