Detailed Description

An implementation of the Map interface which also provides persistance and handling of big amounts of memory. When new object is added, it is persisted into path/mapName, where path defaults to $user.home/.PersistentCachedMaps/ .

This implementation does not support null keys or values, nor duplicate keys. The cacheSize parameter defines how many elements the map will keep in cached memory.
