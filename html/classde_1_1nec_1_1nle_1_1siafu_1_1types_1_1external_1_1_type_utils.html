static String de.nec.nle.siafu.types.external.TypeUtils.stripExternal ( String  publishableClass)
static

The skeleton classes in this library all conform to ORIGINALPACKAGE.external.CLASSNAME, where ORIGINALPACKAGE is the package to which this class would belong in the simulator, as opposed to the package in this external library. In order to compare the classes, we need to strip off "external" and this is precisally what this method does.

Parameters
publishableClassthe name of the class, as external
Returns
the name of the class if it were not external
