Detailed Description

This implementation of SimulatorOutputPrinter prints all the information of an agent, plus the value of the overlays (that is, the whole context set) for each of the users, in a Comma Separated Value file format.

Each time the interval given in the config file has passed, a printout is generated. If the history option is set to true, the printout will accumulate the data for all the iterations. Otherwise only the latest iteration will be kept.

Note that all the fields in the CSV are common to all the entities, and derived from the agent returned by World.getPeople().iterator().next(). Therefore, you need to make sure this particular user has all the entries in his data fields.
