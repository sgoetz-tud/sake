Detailed Description

This class represents a deserialized Place object. It is, however, not a Place object in the Siafu sense of the class, but rather a skeleton representation with the same data fields, but none of the logic.

The purpose of this class is to facilitate dealing with FlatData strings returned by Siafu (using the String constructor), so they can be handled as Java objects.

Notice that the package in this class contains "external" as the last sub package, to avoid erratic behavior should you (mistakenly) add this library to a simulation or a Siafu deployment.
